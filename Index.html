📚 SỔ TAY LÝ THUYẾT CHUYÊN ĐỀ ĐẢO NGỮ
ĐẢO NGỮ SO THAT // SUCH ... THAT
ĐẢO NGỮ SO ... THAT
Trường hợp 1:
Cấu trúc gốc: S₁ + be + so + adj + that + S2 + Verb
👉 Đảo ngữ: So + Adj + be + S1 + that + S2 + Verb + Obj
EX: The sea was so tough that the children could not go swimming.
→ So tough was the sea that the children could not go swimming.
Trường hợp 2:
Cấu trúc gốc: S1 + Verb + so + adv + that + S2 + Verb
👉 Đảo ngữ: So + Adv + trợ động từ + S1 + Verb + that + S2 + Verb + Obj
EX: The children ran so quickly that no one can catch them.
→ So quickly did the children run that no one can catch them.
ĐẢO NGỮ SUCH THAT
Cấu trúc gốc: S1 + be + such + (a/an) + adj + N + that + S2 + Verb
👉 Cách 1: Such + be + (a/an) + N + that + S + V + O
👉 Cách 2: Such + (a/an) + N + be + S + that + S + V + O
EX: It is such a good book that I have read twice.
→ Such is a good book that I have read twice.
→ Such a good book is it that I have read twice.
